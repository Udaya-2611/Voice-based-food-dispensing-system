Voice-Based Food Dispensing System

Description

The Voice-Based Food Dispensing System is an automated solution that dispenses food using voice commands. It uses an Arduino, voice recognition module, and servo motor to release a fixed quantity of food, reducing manual effort and ensuring controlled dispensing.

Features

* Voice-controlled operation
* Automatic food dispensing
* Reduces human effort
* Accurate portion control
* Easy to use system

Components Used

* Arduino UNO
* Voice Recognition Module
* Servo Motor
* LCD 16*2 Display
* Jumper Wires

How It Works

1. The voice module detects predefined commands
2. Arduino processes the command
3. Servo motor rotates to dispense food
4. Motor returns to original position

How to Run

1. Connect all components properly
2. Upload the Arduino code
3. Power the system
4. Give voice command to dispense food
